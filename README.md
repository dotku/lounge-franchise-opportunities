# UNI&CORE San Francisco Franchise Website

A modern, responsive website for UNI&CORE's Korean beauty lounge franchise opportunities in San Francisco.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Beautiful gradient designs with Tailwind CSS
- **Complete Information**: All franchise packages, pricing, and benefits
- **Contact Form**: Easy-to-use contact form for inquiries
- **SEO Optimized**: Proper meta tags and semantic HTML

## Sections

1. **Hero Section**: Eye-catching introduction with key metrics
2. **Market Overview**: San Francisco beauty market analysis
3. **Product Showcase**: Core product lines (Derma10, Derma Home, Hyperloop)
4. **Target Customers**: Ideal franchise partners
5. **Franchise Packages**: Three tiers (Mini Studio, Standard Lounge, Flagship)
6. **Profit Model**: ROI analysis and revenue projections
7. **Founder Club**: Exclusive benefits for first 20 stores
8. **Support Services**: Comprehensive franchise support
9. **Contact Section**: Contact information and inquiry form

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
npm start
```

## Technology Stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **React 19** - UI library
